When multi track recording in the digital domain (computer software or digital recorders) each individual recording track may have multiple audio segments placed throughout the length of the song with periods of nothing in between.


The software used to record the tracks will keep track of the segment placing. If these audio files are used inside a different software recording/playback tool they will most likely not be placed in their correct locations in relation to where they were recorded.


This requires them to be placed manually which can be quite time consuming and prone to errors and timing issues.

To avoid this problem when moving multitrack recordings between platforms, it is common to re-render the full length of each individual track (and its multiple audio clips) into one continuous file.


This process has different names like “Consolidate”, “Bounce”, and “render” depending on the program being used.
